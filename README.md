# Digital Menu Card 

A full stack web application for restaurants to display and manage digital menu.

## Features
- View menu items
- Category wise filtering
- Admin panel to add/edit/delete items
- Responsive design

## Tech Stack
Frontend:
- React.js
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL

## How to run

### Backend
cd backend
node index.js

### Frontend
cd frontend
npm run dev
